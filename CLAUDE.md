## Project

Grocery List Android app — add, delete, and check off purchased items. Data persists via Room database.

## Build & Run

Open the project in Android Studio; it will handle the Gradle wrapper and dependency sync automatically.

```bash
# Build debug APK
./gradlew assembleDebug

# Run tests
./gradlew test

# Install on connected device
./gradlew installDebug
```

> `gradlew` requires the Gradle wrapper JAR (`gradle/wrapper/gradle-wrapper.jar`), which Android Studio generates automatically on first open.

## Tech Stack

- **Language**: Kotlin
- **Min SDK**: 26 / Target SDK: 34
- **Build**: AGP 8.2.2, Gradle 8.6, KSP 1.9.22
- **UI**: ViewBinding, RecyclerView + ListAdapter, Material Components
- **Persistence**: Room 2.6.1 (KSP for code generation)
- **Architecture**: ViewModel + LiveData (AndroidViewModel)

## Architecture

Single-screen MVVM:

```
MainActivity
  └── MainViewModel (AndroidViewModel)
        └── GroceryDao (Room)
              └── GroceryDatabase (singleton)
                    └── GroceryItem (Room entity)
```

- `GroceryItem` — Room entity with `id`, `name`, `isBought` fields
- `GroceryDao` — `getAllItems()` returns `LiveData<List<GroceryItem>>` ordered by `isBought ASC, id DESC` (unchecked items first)
- `GroceryDatabase` — singleton via `companion object`
- `MainViewModel` — exposes `allItems: LiveData`, and `addItem()`, `toggleBought()`, `deleteItem()` as coroutine launchers
- `GroceryAdapter` — `ListAdapter<GroceryItem>` using `DiffUtil`; clears `OnCheckedChangeListener` before setting `isChecked` to avoid re-entrancy; applies strikethrough + alpha on bought items
- `MainActivity` — wires adapter to RecyclerView, observes LiveData, handles keyboard IME action

## Key Conventions

- Room operations run in `viewModelScope` coroutines (all DAO methods are `suspend`)
- KSP (not kapt) is used for Room annotation processing — keep `com.google.devtools.ksp` plugin in sync with the Kotlin version
- `local.properties` (SDK path) is gitignored and generated by Android Studio — do not commit it
